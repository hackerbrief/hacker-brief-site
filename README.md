> Hacker Brief - Developer Blog and Resource Directory

## Build Setup
``` bash
# Install Dependencies
$ npm install

# Serve with Hot-reload at localhost:3000
$ npm run dev

# Build for Production and Launch Server
$ npm run build
$ npm start

# Generate Static Project
$ npm run generate
```
